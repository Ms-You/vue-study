# Vue와 Springboot 연동 프로젝트 #
간단하게 구현한 예술 작품 구매 사이트
## 개발 환경 ##
- Springboot: v3.2.5
- java: 17
- mariaDB
- node.js: v16.15.0
- Vue3
- Vite
- Axios
- Vue Router
- Vuex
## 주요 기능 ##
- Springboot와 Vue3 간에 Axios로 REST API 연동
- 사용자 인증(회원가입, 로그인, 로그아웃)
- 상품 목록 조회
- 상품명으로 상품 검색
- 장바구니에 상품 담기
- 상품 주문
- 주문 내역 조회
